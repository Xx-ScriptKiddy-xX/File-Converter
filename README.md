# File Converter App

<img width="600" alt="File Converter Icon" src="https://github.com/user-attachments/assets/7e233c26-835f-479d-9d68-5865449d8271" />

**File Converter App** is an all-in-one desktop application for converting **images, audio, and video files** with **maximum quality output**. Built with **Python** and **Tkinter**, it features a simple and intuitive GUI and supports batch conversions across multiple media formats.

---

## Key Features

- **Image Conversion** – Supports `jpg, jpeg, png, gif, tga, ico, webp, dds, bmp, heif, svg` with maximum quality.  
- **Audio Conversion** – Supports `mp3, wav, ogg, flac, acc, m4a, wma` and exports at the highest quality.  
- **Video Conversion** – Convert between `mp4, mkv, mov, avi, flv, webm, wmv` at maximum resolution and quality.  
- **Video to Audio** – Extract audio from video files in high-quality formats.  
- **Batch Conversion** – Convert multiple files at once from a folder.  
- **Overwrite Protection** – Prompts before overwriting existing files.  
- **User-Friendly GUI** – Select input/output formats and files with ease.  
- **Maximum Quality Export** – Ensures images, audio, and video are saved at the highest quality possible.  
- **Installer Included** – Comes with a script/executable to set up dependencies like **FFmpeg** automatically.  
  - If **FFmpeg** is already installed, the installer can be skipped.

---

## Built With

- **Python**  
- **Tkinter** – GUI framework  
- **Pillow (PIL)** – Image processing  
- **pydub** – Audio processing  
- **FFmpeg** – Video/audio conversion and processing

---

## Usage

1. Run the installer to set up **FFmpeg** and other dependencies (optional if FFmpeg is already installed).  
2. Launch the app executable.  
3. Select the type of conversion from the **File menu**.  
4. Choose the desired input and output formats.  
5. Select input files or an entire directory.  
6. Click **Convert**, choose an output folder, and let the app handle the rest with maximum quality.
